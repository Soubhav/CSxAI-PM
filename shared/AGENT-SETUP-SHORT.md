# Agent Personalization — Quick Reference

Configurable options when setting up a Master Agent or Sub Agent.

---

## Onboarding Wizard
Guided 3-step setup with live voice preview at each step:
1. **Identity** — Agent name, role, voice profile, accent, language
2. **Personality** — Tone sliders (formality, warmth, vocabulary, humor) with real-time audio comparison
3. **Rules & Guardrails** — Scripts, escalation thresholds, compliance, prohibited topics

After setup: agent created in Draft → test in Sandbox → open in Canvas → Go Live.

---

## Agent Identity
- Agent name, role, and organization branding
- Brand tagline for greetings and closings
- Custom avatar or logo upload

## Voice & Speech
- Voice profile selection with live preview
- Accent and regional voice matching
- Speaking speed, pitch, and pause timing
- Filler words and natural processing cues
- Custom pronunciation overrides for brand terms

## Tone & Personality
- Formality level: formal, balanced, or conversational
- Warmth and empathy intensity
- Vocabulary style: plain language or technical
- Personality keywords to shape agent character

## Language & Localization
- Primary language selection
- Bilingual mid-call language switching
- Regional date, currency, and terminology formats

## Conversation Behavior
- Custom greeting and closing scripts
- Hold behavior: music, narration, or status updates
- Interruption and repeat handling preferences
- Silence timeout with progressive prompts
- Max turns before automatic escalation

## Compliance & Guardrails
- Required disclosure statements
- Prohibited topic restrictions
- Authentication and identity verification rules
- Data masking and handling policies
- Regulatory framework compliance (NCUA, FDIC, RESPA, TCPA)

## Master Agent vs. Sub Agent
- Master Agent sets global defaults for all settings
- Sub Agents inherit and override per scenario
- Independent versioning and sandbox testing
